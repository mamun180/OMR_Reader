User Manual

Introduction

OptiMark Pro-তে স্বাগতম, এটি একটি বিস্তৃত OMR (অপটিক্যাল মার্ক রিকগনিশন) সমাধান যা OMR উত্তরপত্র তৈরি, স্ক্যান এবং পরীক্ষা করার প্রক্রিয়াকে সহজ করার জন্য ডিজাইন করা হয়েছে। এই ম্যানুয়ালটি সফ্টওয়্যারের সমস্ত বৈশিষ্ট্যের জন্য একটি বিস্তারিত নির্দেশিকা প্রদান করে।

Installation

OptiMark Pro একটি একক এক্সিকিউটেবল ফাইল হিসাবে বিতরণ করা হয়। সফ্টওয়্যারটি ইনস্টল করতে, কেবল এই পদক্ষেপগুলি অনুসরণ করুন:

  1. ইনস্টলার ফাইলটি ডাউনলোড করুন (যেমন, OptiMark_Pro_Installer.exe)।
  2. এটি চালানোর জন্য ইনস্টলারে ডাবল-ক্লিক করুন।
  3. ইনস্টলেশন সম্পূর্ণ করতে অন-স্ক্রিন নির্দেশাবলী অনুসরণ করুন।

ইনস্টল হয়ে গেলে, আপনি স্টার্ট মেনু থেকে বা ডেস্কটপ শর্টকাটে ডাবল-ক্লিক করে OptiMark Pro চালু করতে পারেন।

Registration

প্রথমবার চালু হলে, অ্যাপ্লিকেশনটি সীমিত কার্যকারিতা সহ একটি নিবন্ধিত অবস্থায় থাকবে। সমস্ত বৈশিষ্ট্য আনলক করতে, আপনাকে অবশ্যই একটি বৈধ লাইসেন্স কী দিয়ে অ্যাপ্লিকেশনটি নিবন্ধন করতে হবে।

  1. OptiMark Pro চালু করুন। আপনাকে হোম স্ক্রীন দিয়ে স্বাগত জানানো হবে।
  2. "Register Application" বোতামে ক্লিক করুন। এটি আপনাকে Registration ট্যাবে নিয়ে যাবে।
  3. প্রদত্ত ক্ষেত্রে আপনার লাইসেন্স কী লিখুন।
  4. "Register" বোতামে ক্লিক করুন।
  5. যদি লাইসেন্স কী বৈধ হয়, অ্যাপ্লিকেশনটি সমস্ত বৈশিষ্ট্য আনলক করবে এবং আপনি একটি সাফল্য বার্তা দেখতে পাবেন।

দ্রষ্টব্য: লাইসেন্স যাচাইয়ের জন্য একটি ইন্টারনেট সংযোগ প্রয়োজন। লাইসেন্সটি একটি একক মেশিনের সাথে আবদ্ধ।

Template Builder

টেমপ্লেট বিল্ডার OMR শীট টেমপ্লেট তৈরির জন্য একটি শক্তিশালী টুল। একটি টেমপ্লেট আপনার OMR শীটের লেআউটকে সংজ্ঞায়িত করে, যার মধ্যে রয়েছে কোণার মার্কার, বিষয়বস্তু এলাকা এবং উত্তর ও শনাক্তকারীদের জন্য সমস্ত বুদবুদের অবস্থান।

Workflow

  1. Load Image: আপনার OMR শীটের একটি স্ক্যান করা ছবি লোড করতে "Load Image" বোতামে ক্লিক করুন। এই ছবিটি টেমপ্লেট তৈরির জন্য একটি গাইড হিসাবে ব্যবহৃত হবে।
  2. Learn Corner Shape: "1. Learn Corner Shape" বোতামে ক্লিক করুন। অ্যাপ্লিকেশনটি আপনাকে আপনার শীটের কোণার মার্কারগুলির একটির চারপাশে একটি বাক্স আঁকতে অনুরোধ করবে। এটি সফ্টওয়্যারকে স্বয়ংক্রিয়ভাবে কোণগুলি সনাক্ত করতে সহায়তা করে।
  3. Define Content Box: কোণগুলি সনাক্ত হয়ে গেলে, "2. Define Content Box" বোতামে ক্লিক করুন। আপনাকে আপনার OMR শীটের প্রধান বিষয়বস্তু এলাকার চারটি কোণায় ক্লিক করতে অনুরোধ করা হবে। অ্যাপ্লিকেশনটি তখন কোনও স্ক্যানিং বিকৃতি সংশোধন করতে এই এলাকাটিকে "ওয়ার্প" বা सीधा করবে।
  4. Draw ROIs (Regions of Interest): আপনার ওয়ার্পড OMR শীটের বিভিন্ন অংশের উপর আয়তক্ষেত্র আঁকতে "Draw" টুলটি ব্যবহার করুন। এই বিভাগগুলি ছাত্র আইডি, পরীক্ষার কোড বা প্রধান উত্তর বুদবুদের জন্য হতে পারে।
  5. Define ROI Properties:
    • এটিতে ক্লিক করে একটি ROI নির্বাচন করুন।
    • বাম দিকের "ROI Properties" প্যানেলে, আপনি নির্বাচিত ROI-এর বৈশিষ্ট্যগুলি সংজ্ঞায়িত করতে পারেন।
    • Name: ROI-কে একটি বর্ণনামূলক নাম দিন (যেমন, "ছাত্র আইডি", "বিভাগ এ")।
    • Type: ROI-কে একটি প্রকার নির্ধারণ করুন:
      • Identifier: ছাত্র আইডি, রোল নম্বর ইত্যাদির মতো এলাকার জন্য। আপনি বুদবুদের সারি এবং কলামের সংখ্যা, পড়ার ক্রম এবং সংশ্লিষ্ট মানগুলি নির্দিষ্ট করতে পারেন।
      • Answer: প্রধান উত্তর বিভাগগুলির জন্য। প্রশ্নের সংখ্যা, প্রতি প্রশ্নে বিকল্প এবং প্রতিটি বিকল্পের জন্য মানগুলি (যেমন, A, B, C, D) সংজ্ঞায়িত করুন।
      • Qrcode: শীটে যেকোনো QR কোডের জন্য।
      • Annotation: অন্যান্য টীকা বা নোটের জন্য।
  6. Save Template: আপনি সমস্ত ROI সংজ্ঞায়িত করার পরে, আপনার টেমপ্লেটটি একটি JSON ফাইল হিসাবে সংরক্ষণ করতে "Save Template" বোতামে ক্লিক করুন।

Answer Key Scanner

উত্তর কী স্ক্যানার একটি সম্পূর্ণ OMR শীট স্ক্যান করতে এবং এটি একটি উত্তর কী ফাইল হিসাবে সংরক্ষণ করতে ব্যবহৃত হয়। এই ফাইলটি পরে ছাত্র উত্তরপত্র পরীক্ষা করতে ব্যবহৃত হবে।

Workflow

  1. "Answer Key Scanner" ট্যাবে নেভিগেট করুন।
  2. Load Image: সম্পূর্ণ উত্তর কী শীটের একটি স্ক্যান করা ছবি লোড করতে "Load Image" বোতামে ক্লিক করুন।
  3. Automatic Scanning: অ্যাপ্লিকেশনটি স্বয়ংক্রিয়ভাবে কোণগুলি সনাক্ত করতে এবং শীটটি স্ক্যান করতে "মাস্টার টেমপ্লেট" (সেটিংসে সেট করা) ব্যবহার করবে।
  4. Manual Correction (if needed): যদি স্বয়ংক্রিয় স্ক্যান ব্যর্থ হয়, আপনি শীটটি সঠিকভাবে সারিবদ্ধ করতে ম্যানুয়ালি শীটের চারটি কোণায় ক্লিক করতে পারেন।
  5. Adjust Parameters: স্ক্যানের নির্ভুলতা উন্নত করতে চিত্র প্রক্রিয়াকরণ প্যারামিটার (কনট্রাস্ট, উজ্জ্বলতা, ইত্যাদি) সামঞ্জস্য করতে বাম প্যানেলের স্লাইডার এবং নিয়ন্ত্রণগুলি ব্যবহার করুন। পরিবর্তনগুলি দেখতে "Re-Scan" এ ক্লিক করুন।
  6. Verify Answers: স্ক্যান করা উত্তরগুলি ডানদিকের প্যানেলে প্রদর্শিত হবে। আপনি যেকোনো ত্রুটি সংশোধন করতে ম্যানুয়ালি বুদবুদগুলি পরীক্ষা বা আনচেক করতে পারেন।
  7. Save Answer Key: আপনি স্ক্যান করা উত্তরগুলির সাথে সন্তুষ্ট হয়ে গেলে, "Save Answer Key" বোতামে ক্লিক করুন। উত্তর কী একটি JSON ফাইল হিসাবে সংরক্ষণ করা হবে, যার মধ্যে টেমপ্লেট ডেটা, চিত্র সেটিংস এবং সঠিক উত্তরগুলির চূড়ান্ত তালিকা অন্তর্ভুক্ত রয়েছে।

Answer Checker

উত্তর পরীক্ষক অ্যাপ্লিকেশনটির মূল অংশ, যেখানে আপনি এক বা একাধিক উত্তর কী-এর বিপরীতে একাধিক ছাত্র উত্তরপত্র পরীক্ষা করতে পারেন।

Workflow

  1. "Answer Checker" ট্যাবে নেভিগেট করুন।
  2. Add Images: আপনি যে ছাত্র উত্তরপত্র চিত্রগুলি পরীক্ষা করতে চান তা নির্বাচন করতে "Add Images" বোতামে ক্লিক করুন। আপনি একাধিক ফাইল বা একটি সম্পূর্ণ ফোল্ডার নির্বাচন করতে পারেন।
  3. Select Answer Key(s): ড্রপডাউন তালিকা থেকে উপযুক্ত উত্তর কী ফাইল(গুলি) নির্বাচন করুন। আপনি যদি একাধিক উত্তর কী নির্বাচন করেন, অ্যাপ্লিকেশনটি স্বয়ংক্রিয়ভাবে প্রতিটি শীটের জন্য সঠিক কী খুঁজে বের করার চেষ্টা করবে যা মিলিত শনাক্তকারী (যেমন পরীক্ষার কোড বা বিষয়) এর উপর ভিত্তি করে।
  4. Start Scan: "Start Scan" বোতামে ক্লিক করুন। অ্যাপ্লিকেশনটি একে একে চিত্রগুলি প্রক্রিয়াকরণ শুরু করবে।
    • Auto Mode: এই মোডে, অ্যাপ্লিকেশনটি সমস্ত চিত্র স্বয়ংক্রিয়ভাবে প্রক্রিয়াকরণ করার চেষ্টা করবে। যদি এটি একটি শীটে একটি ত্রুটির সম্মুখীন হয়, তবে এটি এড়িয়ে যেতে পারে।
    • Manual Mode: এই মোডে, অ্যাপ্লিকেশনটি প্রতিটি শীটে বিরতি দেবে, যা আপনাকে পরবর্তীটিতে যাওয়ার আগে স্ক্যান করা ডেটা যাচাই করতে এবং সংশোধন করতে দেয়।
  5. Review and Correct: প্রতিটি শীটের জন্য, অ্যাপ্লিকেশনটি সনাক্ত করা উত্তরগুলি হাইলাইট সহ স্ক্যান করা চিত্র প্রদর্শন করে। আপনি ডানদিকের প্যানেলে যেকোনো ভুল সংশোধন করতে পারেন।
  6. Save Results: ফলাফলগুলি স্বয়ংক্রিয়ভাবে একটি এক্সেল ফাইলে সংরক্ষিত হয়। আপনি সেটিংসে আউটপুট ফর্ম্যাট এবং ফাইলের নাম প্যাটার্ন কনফিগার করতে পারেন।

Settings

সেটিংস ট্যাব আপনাকে অ্যাপ্লিকেশনটির বিভিন্ন দিক কনফিগার করতে দেয়।

Default Paths

  • Base Data Directory: প্রধান ফোল্ডার যেখানে সমস্ত অ্যাপ্লিকেশন ডেটা (টেমপ্লেট, উত্তর কী, ফলাফল) সংরক্ষণ করা হবে।
  • Student Data Sheet: ছাত্র তথ্য ধারণকারী একটি ঐচ্ছিক এক্সেল ফাইল। যদি প্রদান করা হয়, এই ডেটা স্বয়ংক্রিয়ভাবে চূড়ান্ত ফলাফলে অন্তর্ভুক্ত করা যেতে পারে।
  • Master Template: উত্তর কী স্ক্যানার এবং উত্তর পরীক্ষক দ্বারা ব্যবহৃত ডিফল্ট টেমপ্লেট ফাইল।

File Naming & Output Patterns

এখানে আপনি আপনার আউটপুট ফাইলগুলির নামকরণের জন্য প্যাটার্ন তৈরি এবং পরিচালনা করতে পারেন। এটি আপনার ফলাফলগুলি সংগঠিত করার একটি শক্তিশালী উপায় সরবরাহ করে। আপনি এর জন্য প্যাটার্ন সংজ্ঞায়িত করতে পারেন:

  • Excel Filename: এক্সেল ফাইলের নাম যেখানে ফলাফলগুলি সংরক্ষিত হয়।
  • Image Renaming: প্রক্রিয়াকরণের পরে স্ক্যান করা চিত্র ফাইলগুলির পুনঃনামকরণ কীভাবে করা উচিত।
  • Answer Key Naming: নতুন উত্তর কীগুলির জন্য ডিফল্ট নামকরণ স্কিম।
