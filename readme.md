# OMR Reader

This is a Python-based OMR (Optical Mark Recognition) reader application with a PyQt6-based graphical user interface.

## Features

*   **Template Builder:** Create templates for OMR sheets.
*   **Answer Key Scanner:** Scan answer keys from O-M-R sheets.
*   **Answer Checker:** Check OMR sheets against answer keys.
*   **Settings:** Configure application settings.
*   **License/Registration:** The application requires a license to unlock all features.

## How to Run

1.  Clone the repository:
    ```
    git clone https://github.com/mamun180/OMR_Reader.git
    ```
2.  Install the required dependencies:
    ```
    pip install -r requirements.txt
    ```
3.  Run the application:
    ```
    python ui_main.py
    ```
