User Manual

Introduction

Welcome to OptiMark Pro, a comprehensive OMR (Optical Mark Recognition) solution designed to streamline the process of creating, scanning, and checking OMR answer sheets. This manual provides a detailed guide to all the features of the software.

Installation

OptiMark Pro is distributed as a single executable file. To install the software, simply follow these steps:

  1. Download the installer file (e.g., OptiMark_Pro_Installer.exe).
  2. Double-click the installer to run it.
  3. Follow the on-screen instructions to complete the installation.

Once installed, you can launch OptiMark Pro from the Start Menu or by double-clicking the desktop shortcut.

Registration

Upon first launch, the application will be in an unregistered state with limited functionality. To unlock all features, you must register the application with a valid license key.

  1. Launch OptiMark Pro. You will be greeted with the Home screen.
  2. Click the "Register Application" button. This will take you to the Registration tab.
  3. Enter your license key in the provided field.
  4. Click the "Register" button.
  5. If the license key is valid, the application will unlock all features, and you will see a success message.

Note: An internet connection is required for license validation. The license is tied to a single machine.

Template Builder

The Template Builder is a powerful tool for creating OMR sheet templates. A template defines the layout of your OMR sheet, including the location of corner markers, content areas, and all the bubbles for answers and identifiers.

Workflow

  1. Load Image: Click the "Load Image" button to load a scanned image of your OMR sheet. This image will be used as a guide for creating the template.
  2. Learn Corner Shape: Click the "1. Learn Corner Shape" button. The application will prompt you to draw a box around one of the corner markers on your sheet. This helps the software identify the corners automatically.
  3. Define Content Box: Once the corners are detected, click the "2. Define Content Box" button. You will be prompted to click on the four corners of the main content area of your OMR sheet. The application will then "warp" or straighten this area to correct for any scanning distortions.
  4. Draw ROIs (Regions of Interest): Use the "Draw" tool to draw rectangles over the different sections of your warped OMR sheet. These sections can be for student IDs, exam codes, or the main answer bubbles.
  5. Define ROI Properties:
    • Select an ROI by clicking on it.
    • In the "ROI Properties" panel on the left, you can define the properties of the selected ROI.
    • Name: Give the ROI a descriptive name (e.g., "Student ID", "Section A").
    • Type: Assign a type to the ROI:
      • Identifier: For areas like student ID, roll number, etc. You can specify the number of rows and columns of bubbles, the order of reading, and the corresponding values.
      • Answer: For the main answer sections. Define the number of questions, options per question, and the values for each option (e.g., A, B, C, D).
      • Qrcode: For any QR codes on the sheet.
      • Annotation: For other annotations or notes.
  6. Save Template: Once you have defined all the ROIs, click the "Save Template" button to save your template as a JSON file.

Answer Key Scanner

The Answer Key Scanner is used to scan a completed OMR sheet and save it as an answer key file. This file will be used later to check student answer sheets.

Workflow

  1. Navigate to the "Answer Key Scanner" tab.
  2. Load Image: Click the "Load Image" button to load a scanned image of the completed answer key sheet.
  3. Automatic Scanning: The application will automatically use the "Master Template" (set in the Settings) to detect the corners and scan the sheet.
  4. Manual Correction (if needed): If the automatic scan fails, you can manually click on the four corners of the sheet to align it correctly.
  5. Adjust Parameters: Use the sliders and controls in the left panel to adjust image processing parameters (contrast, brightness, etc.) to improve the accuracy of the scan. Click "Re-Scan" to see the changes.
  6. Verify Answers: The scanned answers will be displayed in the panel on the right. You can manually check or uncheck the bubbles to correct any errors.
  7. Save Answer Key: Once you are satisfied with the scanned answers, click the "Save Answer Key" button. The answer key will be saved as a JSON file, which includes the template data, the image settings, and the final list of correct answers.

Answer Checker

The Answer Checker is the core of the application, where you can check multiple student answer sheets against one or more answer keys.

Workflow

  1. Navigate to the "Answer Checker" tab.
  2. Add Images: Click the "Add Images" button to select the student answer sheet images you want to check. You can select multiple files or an entire folder.
  3. Select Answer Key(s): Select the appropriate answer key file(s) from the dropdown list. If you select multiple answer keys, the application will automatically try to find the correct key for each sheet based on matching identifiers (like exam code or subject).
  4. Start Scan: Click the "Start Scan" button. The application will begin processing the images one by one.
    • Auto Mode: In this mode, the application will attempt to process all images automatically. If it encounters an error on a sheet, it may skip it.
    • Manual Mode: In this mode, the application will pause on each sheet, allowing you to verify the scanned data and make corrections before moving to the next one.
  5. Review and Correct: For each sheet, the application displays the scanned image with the detected answers highlighted. You can correct any mistakes in the right-hand panel.
  6. Save Results: The results are automatically saved to an Excel file. You can configure the output format and filename pattern in the Settings.

Settings

The Settings tab allows you to configure various aspects of the application.

Default Paths

  • Base Data Directory: The main folder where all application data (templates, answer keys, results) will be stored.
  • Student Data Sheet: An optional Excel file containing student information. If provided, this data can be automatically included in the final results.
  • Master Template: The default template file to be used by the Answer Key Scanner and Answer Checker.

File Naming & Output Patterns

Here you can create and manage patterns for how your output files are named. This provides a powerful way to organize your results. You can define patterns for:

  • Excel Filename: The name of the Excel file where the results are saved.
  • Image Renaming: How the scanned image files should be renamed after processing.
  • Answer Key Naming: The default naming scheme for new answer keys.

You can use components like the current date, time, or the values from scanned identifier ROIs (e.g., <Exam_Code>_<Subject>.xlsx) to create dynamic filenames.
